# mask-Detector
python script for mask detection


this script require these Python Libraries

 - tensorflow>=1.15.2
 - keras==2.3.1
 - imutils==0.5.3
 - numpy==1.18.2
 - opencv-python==4.2.0.*
 - matplotlib==3.2.1
 - scipy==1.4.1
  
  ## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install -r requirements.txt
```
